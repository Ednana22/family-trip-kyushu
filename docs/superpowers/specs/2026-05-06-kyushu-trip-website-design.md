# 九州家族旅行網站 — 設計規格文件

**日期：** 2026-05-06  
**專案：** Kyushu Family Trip Website  
**受眾：** Edna（生活美學 influencer，6,000 粉絲）× 小姑（親子學習團購主，15,000 粉絲）的粉絲群

---

## 1. 目標

製作一個可分享給粉絲的九州家族旅遊紀錄網站，類似 Google Maps 概念，包含：
- 每天地標標示（地點、營業時間）
- 行程路線地圖
- 兩人的旅遊筆記（推不推薦、可以買什麼等）

---

## 3. 設計風格

### 整體美感
- **參考：** Airbnb × Apple — 大量留白、卡片式佈局、克制用色
- **背景：** 純白 `#ffffff`
- **重點色：** `#748573`（鼠尾草綠）— 僅用於圖釘、tag 框線、筆記左側線、Day 標題、hover 效果
- **色塊原則：** 避免大面積色塊，顏色只給重點細節

### Tag 樣式
- 只用外框線（`border: 0.5px solid`），無填色背景
- 推薦 tag：`#748573` 綠色外框 + 綠字
- 其他 tag（如「必買」）：灰色外框 + 灰字

### 筆記作者頭像
- E（Edna）和 S（小姑）統一樣式：淺灰底 `#f0f0f0` + `#748573` 綠字
- 圓形頭像，尺寸 16×16px（手機內）

## 5. 地標卡片結構

每張卡片包含：
1. **照片區**：灰色佔位框，支援上傳圖片（`<input type="file">`），點擊可更換
2. **地標名稱**：font-weight 500
3. **營業時間**：灰色小字
4. **Tag 列**：outline 風格，推薦 / 必買 / 親子友善 等
5. **筆記區**：
   - 每則筆記一行，左側圓形頭像（E 或 S）
   - 筆記文字用灰底泡泡框呈現
   - 左側有 `#748573` 1.5px 線條點綴

---

## 6. 技術規格

| 項目 | 規格 |
|------|------|
| 技術 | 單一 `index.html` 檔案（HTML + CSS + JS） |
| 地圖 | Google Maps JavaScript API |
| 響應式 | RWD，mobile-first，breakpoint 768px |
| 字型 | System font stack（-apple-system, BlinkMacSystemFont...） |
| 照片 | `<input type="file" accept="image/*">`，FileReader 預覽，存於 localStorage |
| 外部依賴 | Google Maps API（需申請 Key） |
| 部署 | 靜態 HTML，可上傳至 Netlify / GitHub Pages 分享連結 |

---

## 7. Google Maps API 注意事項

- 需申請 Google Cloud 帳號 + 啟用 Maps JavaScript API
- 建議設定每日用量上限（Daily quota limit）避免超額收費
- API Key 需限制來源網域（Referrer restriction）保護安全
- 免費額度：每月 $200 美金額度，一般流量足夠
